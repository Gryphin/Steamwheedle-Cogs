# roster
Base command for handling scheduling of tasks

## Usage
All settings are available under `[p]roster`.

## Guild Commands
| Command        | Description                                                  |
| :------------- | :----------------------------------------------------------- |
| `[p]roster`   | Displays the list of current guilds.                          |  

### Future Features
`[p]roster ...`
| Command        | Description                                                  |
| :------------- | :----------------------------------------------------------- |
| `url`          | Changes the url of the spreadsheet, if compromised           |
