# deck
Base command for decoding & displaying Warcraft Rumble deck codes

## Usage
All settings are available under `[p]deck`.

## Guild Commands
| Command        | Description                                                  |
| :------------- | :----------------------------------------------------------- |
| `[p]deck`   | Displays the decoded deck.         |  

### Future Features
`[p]deck ...`
| Command        | Description                                                  |
| :------------- | :----------------------------------------------------------- |
| `url`          | Changes the url of the spreadsheet, if compromised           |
