# Installation
### Recommended - Built-in Downloader
```
[p]repo add Steamwheedle-Cogs https://github.com/Gryphin/Steamwheedle-Cogs
[p]cog install Steamwheedle-Cogs <cogname>
[p]load <cogname>
```
